Clip any content from a website into your Obsidian.


## How to start

Step1: Install the `Local REST API` plugin in Obsidian: https://coddingtonbear.github.io/obsidian-local-rest-api/

Step2: download this repo to local, open chrome's extension page, and select the `Load Unpacked extension`, then select the folder you download, done.

## How to use

Config the profile for spicifal website or set a common profile for all website. `URL`、`DATE`、`TIMESTAMEP` is the build-in variables, not need to get from the css selector or js.
