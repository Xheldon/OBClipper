## Manual Install

1. Go to the [GitHub Releases](https://github.com/Xheldon/OBClipper/releases) page
2. Download the latest `.zip` file
3. Unzip it to a local folder
4. Open `chrome://extensions/` in Chrome
5. Enable **Developer Mode** (toggle in the top-right corner)
6. Click **Load unpacked** and select the unzipped folder
7. Done! The OBClipper icon will appear in your toolbar

## Requirements

- Chrome (or any Chromium-based browser: Edge, Arc, Brave, etc.)
- [Obsidian](https://obsidian.md) with the [Local REST API](https://github.com/coddingtonbear/obsidian-local-rest-api) plugin
