# 下载

## Chrome Web Store

> 即将上线 — 插件将发布到 Chrome Web Store。

## 手动安装

1. 前往 [GitHub Releases](https://github.com/Xheldon/OBClipper/releases) 页面
2. 下载最新的 `.zip` 文件
3. 解压到本地文件夹
4. 在 Chrome 中打开 `chrome://extensions/`
5. 开启右上角的**开发者模式**
6. 点击**加载已解压的扩展程序**，选择解压后的文件夹
7. 完成！OBClipper 图标将出现在工具栏中

## 系统要求

- Chrome（或任何基于 Chromium 的浏览器：Edge、Arc、Brave 等）
- [Obsidian](https://obsidian.md) 并安装 [Local REST API](https://github.com/coddingtonbear/obsidian-local-rest-api) 插件
