# 自动双链

自动双链功能会将剪藏内容中出现的 Vault 文件名自动转换为 Obsidian `[[双链]]`。

## 工作原理

当你剪藏页面时，OBClipper 会：

1. 通过 Local REST API 获取你 Vault 中的文件名列表
2. 扫描剪藏的 Markdown 内容，查找匹配的文件名
3. 将每个匹配项的**首次出现**替换为 `[[双链]]`

这让你的剪藏笔记自动融入知识图谱，无需手动操作。

## 配置

在设置 → **自动双链** 中：

- **启用** — 开关此功能
- **排除文件夹** — 逗号分隔的 Vault 文件夹列表（如 `Templates, Archive/old`）

排除的文件夹中的文件不会被链接，可以减少模板文件或归档内容带来的干扰。

## 示例

如果你的 Vault 中有一个名为 `Obsidian.md` 的文件，而你剪藏的页面提到了「Obsidian」，剪藏内容中将出现 `[[Obsidian]]` 而非纯文本。
